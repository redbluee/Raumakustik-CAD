This documentation provides an overview over the functionalities from RoomAcousticWizard, a tool designed to calculate and optimize reverberation times in rooms according to the DIN 18041 [DIN18041]. Included are instructions on how to use the tool, both through a graphical user interface (GUI) and programmatically. The internal program logic is also documented, providing insights into the GUI, the calculation methods and algorithms used.
